A playful Hugo theme inspired by Japanese web design aesthetics. brennan.jp.net recreates the compact, text-heavy, colorful style of traditional Japanese websites while maintaining modern accessibility and performance standards.

Key features:
- **Japanese web design aesthetic**—vibrant colors, dense layouts, thick borders
- **Retro web elements**—webrings, 88x31 badges, visitor counters
- **Modern accessibility**—semantic HTML, keyboard navigation, screen reader support
- **Customizable colors**—simple configuration for primary and secondary color schemes
- **Toggle sidebar**—flexible layout options
- **CJK character support**—optimized for Japanese, Chinese, Korean text
- **Easy configuration**—no coding required for basic customization

Perfect for those who miss the creative, playful web of the past while needing modern functionality and accessibility.
