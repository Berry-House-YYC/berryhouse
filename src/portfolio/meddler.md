A Medium export converter that helps you reclaim your content. Meddler transforms Medium's HTML export into clean, portable Markdown files ready for any static site generator.

Key features:
- **Dual interface**—web app for beginners, CLI for power users
- **Multiple SSG support**—Hugo, Eleventy, Jekyll, Astro, or generic formats
- **Flexible front matter**—YAML, TOML, or JSON output options
- **Image handling**—download locally or keep CDN links
- **Embed conversion**—YouTube, GitHub Gists, Tweets to native shortcodes
- **Complete data extraction**—posts, drafts, responses, bookmarks, claps, highlights
- **Privacy-first**—web version runs entirely in your browser
- **Open source**—AGPL-3.0 license, contributions welcome

Perfect for writers looking to migrate from Medium to their own independent blog while preserving years of content and engagement data.
