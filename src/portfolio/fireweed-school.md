A simple and elegant landing page for Fireweed School, built with Hugo and custom CSS. The site embodies the school's educational mission with a clean, accessible design that works across all devices.

Key features:
- **Simple Hugo structure**—fast, secure static site
- **Custom CSS design**—tailored styling without frameworks
- **Fully responsive**—works perfectly on desktop, tablet, and mobile
- **Fast loading**—minimal dependencies, optimized performance
- **Accessible by default**—semantic HTML, proper contrast ratios
- **Easy to maintain**—simple content structure and clear organization
- **SEO optimized**—clean markup and fast performance

Perfect for educational institutions and organizations that need a professional web presence without complexity.
